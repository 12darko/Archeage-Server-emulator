Процесс отладки
Через версию Microsoft Visual Studio 2015+
После редактирования измените ArcheAgeLogin и ArcheAge
Настройте MYSQL, учетную запись: root, password: root и импортируйте следующий sql-файл ArcheAgeLogin
Номер установки Glyph platform archeage Откройте корневой каталог программного обеспечения Glyph \ Library \ GlyphLibrary.xml и измените IP-адрес примерно 178 строк на 127.0.0.1 и установите файл в режим только для чтения (чтобы восстановить просто отключить режим только для чтения или прямой Удалить файл)
Затем вы можете продолжить мою работу
